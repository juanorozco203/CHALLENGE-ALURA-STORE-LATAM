# CHALLENGE-ALURA-STORE-LATAM
Proyecto realizado por lo aprendido en el curso "Fundamentos de Python y Datos G9 - ONE" de Alura Latam &amp; Oracle
Este proyecto es el primer challenge de Data Science de Alura Latam. El objetivo es ayudar al Sr. Juan a decidir cuál de las 4 tiendas de su cadena Alura Store debe vender para iniciar un nuevo emprendimiento.
Para ello se analizaron datos reales de ventas de las 4 tiendas, evaluando los siguientes factores:

Ingresos totales por tienda
Ventas por categoría de producto
Calificación promedio de los clientes
Productos más y menos vendidos
 Costo de envío promedio por tienda

Al final del análisis se presenta una recomendación basada en los datos sobre qué tienda debe vender el Sr. Juan.

Tecnologías Usadas

Python 3
Pandas - carga y manipulación de datos
Matplotlib - visualización de datos
Google Colab - entorno de desarrollo

 Cómo Ejecutarlo

Abre el archivo AluraStore_Analisis.ipynb en Google Colab
Ejecuta la primera celda para cargar los datos automáticamente desde GitHub
Ejecuta las celdas en orden de arriba hacia abajo
Al final encontrarás el informe con la recomendación final

No es necesario descargar ningún archivo CSV, los datos se cargan directamente desde la URL.
